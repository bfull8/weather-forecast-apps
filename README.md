# Weather Forecast Apps

A multi-version weather forecast application that evolved through **four different technology stacks** over time. This repository showcases the progression from a basic web app to a modern cross-platform mobile application.

## 📖 Project Journey

This project started as a simple web application using Google Maps and has since been rebuilt using modern frameworks and native technologies.

| Version | Technology | Folder |
|---------|------------|--------|
| **v1** | HTML, CSS, JavaScript + Google Maps API | [`/legacy-web`](./legacy-web) |
| **v2** | React (Web) | [`/react-web`](./react-web) |
| **v3** | Native iOS (Swift / SwiftUI) | [`/native-ios`](./native-ios) |
| **v4** | React Native | [`/react-native`](./react-native) |

---

## 🛠️ Tech Stack Evolution

- **Frontend Evolution**: Vanilla JS → React → SwiftUI → React Native
- **Mapping**: Google Maps API (early versions) → Map integration in React Native
- **State Management**: Progressed from basic to modern solutions
- **Architecture**: Learned better separation of concerns and scalability
